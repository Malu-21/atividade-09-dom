# 🔍 Encontre o Diferente

## 📌 Sobre o jogo
Este é um jogo desenvolvido com JavaScript puro (DOM), onde o jogador deve encontrar o emoji diferente o mais rápido possível dentro de um tempo limite.

---

## 🎮 Como jogar
- Digite seu nome:
- Clique em "Jogar"
- Encontre o 😎 entre os 😀
- Cada acerto vale +10 pontos
- Cada erro tira -3 pontos
- Você tem 30 segundos

---

## 🧠 Mecânica usada
- Manipulação do DOM
- Eventos de clique
- setInterval (cronômetro)
- Math.random (posição aleatória)
- innerHTML para atualização da tela

---

## ⭐ Diferencial do meu jogo
- Sistema de tempo (30s)
- Pontuação dinâmica
- Botão de reiniciar no final
- Mensagem final com nome do jogador
- Favicon com emoji na aba do navegador

---

## 🐞 Bug que eu resolvi
No início tive problema com o botão "Jogar novamente" aparecendo antes da hora, mas resolvi removendo ele do HTML e criando apenas no final do jogo via JavaScript.

---

## 📚 O que aprendi
Aprendi a manipular o DOM, criar elementos dinamicamente e controlar eventos e tempo em JavaScript.
